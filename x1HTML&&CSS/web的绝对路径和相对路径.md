HTML 属性值**引用文件路径**的一些通用规则：
-   同级可简写文件名，例如：`my-image.jpg`。
-   同级 `./` 上级 `../`
-   Windows 的文件系统使用反斜杠 `\`，但在 html 使用的是正斜杠 ` / `。

```
     /    根路径
    
    ./     同级路径
    
    ../    上一级路径
```
