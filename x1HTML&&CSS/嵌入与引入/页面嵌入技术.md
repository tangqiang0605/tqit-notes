## 概念
在 HTML 中除了文字等简单的页面内容，其他多媒体内容比如图片、视频、音乐、画板、PDF 阅读器、H5 游戏等功能，都是通过页面嵌入技术实现的。为我们的页面带来了更多精彩的内容。

早期，pdf、svg、flash 可以通过 object 标签或 embed 标签嵌入。存在很多安全问题，现已过时，不再支持。

现在，iframe、canvas、video 出现了。而 iframe 提供嵌入网页的方法。
