## 文字遮罩
用文字的形状去裁剪图片。
```
.logo {
  border: 1px solid black;
  font-size: 200px;
  font-weight: 900;
  padding: 20px;
  aspect-ratio: 1;
  display: flex;
  align-items: center;
  background: url(./c5366469b00940f5b8077c20c2c2d089.jpg) center center no-repeat;
  background-size: 100% 100%;

  -webkit-background-clip: text;
  color: transparent;
}
加在文字上
```
