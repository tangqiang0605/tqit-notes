## 引入
在 vue 中，background 使用 url 相对引用 assets 是可以的。

## 渐变叠加
渐变在 background 中的本质是图片。在 background 中，图片可以设置位置和叠加，因此渐变也可以设置位置和叠加。

```
设置图像位置
background: pic1 left top, pic2 right top;
background-repeat: no-repeat;
```
