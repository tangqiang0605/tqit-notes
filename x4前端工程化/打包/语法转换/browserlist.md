browserslist: 设置项目的浏览器兼容情况。

```
browserslist
{
  "browserslist": [
    "> 1%",
    "last 2 versions"
  ]
}
```
支持的工具：

Autoprefixer
Babel
postcss-preset-env
eslint-plugin-compat
stylelint-no-unsupported-browser-features
postcss-normalize
详细参考 browserslist.
