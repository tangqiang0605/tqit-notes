```js
{

  "link": {

    "description": "5 minutes to learn the most important Docusaurus concepts."
```

## 文档
yaml配置，要遵循yaml语法
```js
---
排序
sidebar_position: 1
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url路由
---
```
